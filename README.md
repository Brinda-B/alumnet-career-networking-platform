# AlumNet - Career Networking Platform

## Overview
AlumNet is a full-stack alumni networking platform designed to connect students and alumni for mentorship, networking, and career opportunities.

## Features
- Student, Alumni, and Admin roles
- Alumni Directory
- Real-time Chat System
- Job & Internship Portal
- Notifications
- Profile Management
- Admin Approval Dashboard

## Technology Stack
- React.js
- Vite
- Supabase
- PostgreSQL
- JavaScript
- HTML/CSS

## Future Enhancements
- Video Mentorship Sessions
- AI-Based Career Recommendations
- Event Management System
- Mobile Application Support
