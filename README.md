# AlumNet - Career Networking Platform

## Overview
AlumNet is a full-stack alumni networking platform that connects students and alumni for networking, mentorship, and career opportunities.

## Features
- Student, Alumni, and Admin roles
- Alumni Directory
- Real-time Chat
- Job & Internship Portal
- Notifications
- Profile Management

## Technology Stack
- React.js
- Supabase
- PostgreSQL
- JavaScript
- HTML/CSS
